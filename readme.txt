Forwards mobile users to a specific website using either JavaScript or PHP.

== Description ==

This plugin allows you to specify a different website for mobile users. When a mobile user loads up the WordPress website they will automatically be forwarded
to that address.

You can also specify if you want to forward mobile users using JavaScript or PHP.

== Installation ==

1. Upload `mobile_re-direct.php` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the `Plugins' menu in WordPress.
3. On the 'Mobile Re-direct' settings page specify where you want mobile users to go and set the forwarding as `JavaScript` or `PHP`.
